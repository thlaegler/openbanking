This is the API

createInternationalPaymentConsents

URL

https://localhost:8080/open-banking/v2.3/international-payment-consents

HTTP Method

POST

Response Type

OBWriteInternationalConsentResponse3

Parameters

  • Body: OBWriteInternationalConsent3Param OBWriteInternationalConsent3(OBWriteInternationalConsent3)

    Default

  • Header: Authorization String

    An Authorisation Token as per https://tools.ietf.org/html/rfc6750

  • Header: x-idempotency-key String

    Every request will be processed only once per x-idempotency-key. The Idempotency Key will be valid for 24 hours.

  • Header: x-jws-signature String

    A detached JWS signature of the body of the payload.

  • Header: x-fapi-auth-date String

    The time when the PSU last logged in with the TPP. All dates in the HTTP headers are represented as RFC 7231 Full Dates. An example is below: Sun, 10 Sep 2017 19:43:31 UTC

  • Header: x-fapi-customer-ip-address String

    The PSU's IP address if the PSU is currently logged in with the TPP.

  • Header: x-fapi-interaction-id String

    An RFC4122 UID used as a correlation id.
