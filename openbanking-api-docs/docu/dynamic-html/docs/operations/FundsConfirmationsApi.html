createFundsConfirmationConsents

URL

https://localhost:8080/open-banking/v2.3/funds-confirmation-consents

HTTP Method

POST

Response Type

OBFundsConfirmationConsentResponse1

Parameters

  • Body: OBFundsConfirmationConsent1Param OBFundsConfirmationConsent1(OBFundsConfirmationConsent1)

    Default

  • Header: Authorization String

    An Authorisation Token as per https://tools.ietf.org/html/rfc6750

  • Header: x-fapi-auth-date String

    The time when the PSU last logged in with the TPP. All dates in the HTTP headers are represented as RFC 7231 Full Dates. An example is below: Sun, 10 Sep 2017 19:43:31 UTC

  • Header: x-fapi-customer-ip-address String

    The PSU's IP address if the PSU is currently logged in with the TPP.

  • Header: x-fapi-interaction-id String

    An RFC4122 UID used as a correlation id.
